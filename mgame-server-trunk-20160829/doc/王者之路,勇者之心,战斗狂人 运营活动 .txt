1 王者之路

config 
	bonusData		{}		奖励数据
		index		int		连胜场次[1+ 当超过最大值 取最大值]
		value		int		钻石数量/荣誉点数
	
selfData
	takeState		int		领取状态 1 未领取 不能领奖 2 未领取 能领奖 3 已领取 不能再领奖
	conWinCount		int		连胜场次
	
凌晨处理selfData
	takeState		1 
	conWinCount		0
	
连胜20场 发勋章 这个不应该放在活动类 是一个具体的功能

打完一场 
胜利 conWinCount + 1
失败 takeState 1

领取协议
前提 takeState状态为2 连胜记录大于1 未领取状态
过程 takeState 3
